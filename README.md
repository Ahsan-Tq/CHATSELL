# Chatsell

WhatsApp automation for small Pakistani sellers. Auto-replies to customer questions, captures orders through conversation, and gives sellers one dashboard to manage everything.

---

## The Problem

Small sellers on Instagram and WhatsApp are losing sales not because their product is bad — but because they can't respond fast enough. A customer asks "price?" and if there's no reply in 5 minutes, they move on. These sellers are copy-pasting the same answers 50 times a day and losing orders every time they go offline.

---

## What Chatsell Does (v1 MVP)

- Auto-replies to common questions — price, availability, delivery, location
- Order capture — bot collects name, address, and product via WhatsApp conversation
- Seller dashboard — one place to see all incoming orders and chats

## What v1 Does NOT Do

No follow-ups, no broadcasts, no analytics, no AI smart replies. Those are v2.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Backend | Python, FastAPI |
| Frontend | React, Tailwind CSS |
| Database | Supabase |
| Hosting | Railway (backend), Vercel (frontend) |
| Messaging | Meta WhatsApp Cloud API |
| AI | Grok API |

---

## Target Users

Small sellers in Pakistan — Instagram shops, home-based food businesses, boutiques — operating through WhatsApp DMs. No technical knowledge required to use this.

---

## Roadmap

- **v1** — Auto-replies, order capture, seller dashboard
- **v2** — Follow-ups, broadcast messages, analytics, smart AI replies
